# Qcloud 工程使用示例

## 配网方式
支持多种配网方式连接到腾讯云，例如 airkiss/softap/smartconfig以及直连等方式。

配网的示例代码参见 `app/usr_app.c`。

腾讯云物联网开发平台对当前产品也需要作相同的配网方式，例如物联网开发平台设置产品交互方式为 **airkiss** ，那么工程代码也需要设置为 **airkiss** 配网。
